sbt-scalafmt
============

Welcome to sbt-scalarifmt, an <a href="https://github.com/sbt/sbt">sbt</a> plugin adding support for source code formatting using <a href="http://scalafmt.org">scalafmt</a>.

Documentation on how to use scalafmt from sbt can be found in the <a href="https://olafurpg.github.io/scalafmt/#sbt">scalafmt docs</a>.
